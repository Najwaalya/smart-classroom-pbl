# Smart Classroom Monitoring System

Dashboard monitoring kelas berbasis IoT untuk mendeteksi kehadiran dan aktivitas secara real-time.

## 🚀 Features
- Real-time people counting
- Status kelas otomatis (Kosong / Tidak Pasti / Ada Perkuliahan)
- Monitoring sensor:
  - Infrared (orang masuk/keluar)
  - PIR (deteksi gerakan)
  - Suhu ruangan
- Dashboard interaktif & responsif

## 🛠 Tech Stack
- Next.js
- Tailwind CSS
- IoT (ESP32)
